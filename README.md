# AirQualityMonitoringSystem
This is a WebApp made for monitoring Air Quality made for Internship project under TATA CONSULTANCY SERVICES.

### Features :
- User/Admin Login
- Admin Panel
  - Add AQI Details of a region 
  - Add new region
  - View air quality
- Notifications of high AQI in a region
